# Mistralody

Next.js Strudel workspace with chat and voice agent controls, AI-powered music edits, lyric generation, and ElevenLabs vocal synthesis.

## Run

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Environment

Create `.env`:

```bash
MISTRAL_API_KEY=...
ELEVENLABS_API_KEY=...
NEXT_PUBLIC_APP_NAME=Mistralody
```
